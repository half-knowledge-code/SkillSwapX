# 🚀 SkillSwapX

### Exchange Skills, Grow Together 💡

SkillSwapX is a modern full-stack web application that allows users to **exchange skills without money** and grow through **honest, anonymous feedback**.

---

## 🌟 Features

✨ Skill Exchange System (Teach & Learn)
💬 Real-time Chat (Socket.io)
🤝 Smart User Matching
⭐ Anonymous Feedback System
🏆 Rating & Reputation

---

## 🧠 Problem It Solves

Many people want to learn new skills but:

* ❌ Cannot afford courses
* ❌ Lack proper guidance
* ❌ Don’t receive honest feedback

👉 SkillSwapX solves this by enabling **peer-to-peer learning without financial barriers**

---

## 💡 How It Works

1. Create your profile
2. Add skills you can teach & learn
3. Get matched with users
4. Chat and exchange knowledge
5. Give anonymous feedback

---

## 🛠️ Tech Stack

**Frontend:** React.js
**Backend:** Node.js + Express
**Database:** MongoDB
**Real-time:** Socket.io
**Auth:** JWT

## 🚀 Getting Started

```bash
git clone https://github.com/half-knowledge-code/SkillSwapX.git
cd frontend && npm install
cd ../backend && npm install
```

Create `.env` in backend:

```
PORT=5000
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret
```

Run project:

```bash
# backend
cd backend
npm run dev

# frontend
cd frontend
npm run dev
```
